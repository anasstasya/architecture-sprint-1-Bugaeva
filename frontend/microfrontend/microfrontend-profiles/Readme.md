# Микрофронтенд profiles

Данный микрофронтенд предназначен для управления профилями пользователей.

Экспортируются компоненты:

  * `UserPane` — область просмотра информации о пользователе (аватар, данные
    профиля с возможностью редактирования

Генерируются события:

  * `profiles-user-update` — профиль пользователя обновлён.
    Детали события — структура User со следующими полями:
      * `_id` — id пользователя
      * `name` — имя пользователя
      * `about` — информация о пользователе
      * `avatar` — URL аватара
  * `_profiles-edit-profile-popup-open` — служебное событие на открытии
    диалога редактирования профиля
  *  `_profiles-edit-avatar-popup-open` — служебное событие на открытии
    диалога редактирования URL аватара
